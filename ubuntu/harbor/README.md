# [Harbor](https://goharbor.io/docs/2.9.0/install-config/installation-prereqs/)
쿠버네티스를 위한 레포지토리

## Prerequisites
- Hardware: 
```
Minimum: 2 CPU / 4GB Memory / 40GB Disk
Recommended: 4 CPU / 8GB Memory / 160GB Disk
```
- Software
  - Docker Engine
  - Docker Compose
  - OpenSSL

- Ports
```
HTTPS 443
HTTPS 4443
HTTP 80
```
